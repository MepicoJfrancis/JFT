üìö My First Web Development Project

Posted by Francis on April 8, 2025

Creating my first portfolio website has been an exciting challenge! I started with basic HTML structure and gradually added CSS styling to make it visually appealing.

One thing I learned was how to properly use HTML entities. Did you know you can display special characters like & (ampersand) or < (less than) using codes? It's essential for showing code examples or symbols that might otherwise be interpreted as HTML tags.

While I enjoy the technical aspects of coding, I've also developed a passion for design. Here's what I've learned about creating visually appealing web pages:
